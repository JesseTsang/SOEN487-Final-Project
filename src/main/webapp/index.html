ABOUT THE PROJECT

Key features of our project

Retailer

A mimic e-commerce web site that process customer's order, a REST call on getCatalog() to asynchronously fetch catalog information.

Warehouse

Three warehouse instances ready to take requests from retailer. Communication between retailer and warehouse with SOAP. Replenish function is call when certain stock is below threshold.

Manufacturer

Three manufacturer instances taking request from warehouses to replenish stock. Communication between retailer and warehouse with SOAP.

Key Technology Used

  • Java 7/Javascript/SOAP/RESTful/HTML
  • NetBeans/OpenESB
  • MySQL
